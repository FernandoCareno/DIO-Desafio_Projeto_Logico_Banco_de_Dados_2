# Projeto Banco de Dados – Oficina Mecânica

Projeto desenvolvido para criação do esquema lógico de um banco de dados para uma oficina mecânica.

O sistema contempla:

- Clientes
- Veículos
- Mecânicos
- Ordens de Serviço
- Serviços
- Peças

Foram implementados:

- Script de criação do banco
- Script de inserção de dados
- Queries com SELECT, WHERE, JOIN, ORDER BY e HAVING

Banco utilizado: MySQL
